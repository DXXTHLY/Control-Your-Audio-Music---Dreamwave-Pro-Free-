## 📦 Installation

DreamWave Pro is distributed as a `.crx` file. It takes about 60 seconds to install.

### Step 1 — Download the file

Download **DreamWave.crx** from the releases section of this page and save it somewhere you can find it (Desktop is fine).

---

### Step 2 — Open your browser's extension page

| Browser | Address to type |
|---------|----------------|
| Chrome | `chrome://extensions` |
| Edge | `edge://extensions` |
| Opera | `opera://extensions` |

Paste that into your address bar and hit Enter.

---

### Step 3 — Enable Developer Mode

In the **top right corner** of the extensions page, toggle on **"Developer Mode"**.

> ⚠️ This is required to load `.crx` files that aren't from the Web Store. It does not make your browser less safe.

---

### Step 4 — Drag and drop the file

Drag the **DreamWave.crx** file you downloaded directly onto the extensions page and drop it.

A prompt will appear asking you to confirm — click **"Add Extension"**.

---

### Step 5 — You're in

DreamWave Pro will appear in your extensions list. Click the puzzle piece icon 🧩 in your browser toolbar and pin DreamWave so it's always one click away.

## 🚀 How to Use

1. **Go to any site with audio or video** — YouTube, Twitch, Spotify, Netflix, etc.
2. **Click somewhere on the page first** before opening DreamWave. The browser requires a click on the page before audio processing can activate.
3. **Click the DreamWave icon** in your toolbar to open the popup.
4. **Adjust your sliders** — changes apply instantly, no Apply button needed (though one exists if you want to force a send).
5. **Hit Disable** to bypass all effects and return to normal audio at any time.

## ❓ FAQ

**Does it work on Firefox?**  
Not currently. Chrome, Edge, and Opera only.

**Why do I need to click the page before it works?**  
Browsers require a user gesture (a click) before allowing audio processing to start. This is a browser security rule, not a DreamWave limitation. Just click anywhere on the video or page before opening DreamWave.

**The reverb slider doesn't seem to do anything.**  
Reverb and Wet Mix work together. Reverb controls the size/intensity of the effect, Wet Mix controls how much of it you hear. Raise both for the best result. If you drag Reverb up from zero, Wet Mix will auto-adjust so you hear it immediately.

**Can I use it on multiple computers?**  
Open a ticket in Discord and we can sort it out.

**What happens when my Pro license expires?**  
Free features (Speed, Volume, Reverb, 10-Band EQ, Genre Presets) continue working. Pro features lock until you renew.

**Is there a free version?**  
Yes — installing the extension gives you access to all free features immediately with no key required.

**Will this slow down my browser?**  
DreamWave only activates on tabs where you've interacted with it. It uses the browser's native Web Audio API so performance impact is minimal.
